# Crypto Data Dashboard Service
Mock crypto data provider service for crypto-data-dashboard.

## Prerequisites

- Node.js 18+
- npm 9+

## Getting Started

### Install dependencies

```bash
npm install
```

### Run in development mode

```bash
npm run dev
```

The server starts at `http://localhost:3001`. Nodemon watches the `src/` folder and restarts automatically on any file change.

### Build for production

```bash
npm run build
```

Compiles TypeScript to JavaScript in the `dist/` folder.

### Run the production build

```bash
npm start
```
